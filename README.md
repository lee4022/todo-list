오늘의 할 일을 관리하는 심플한 웹 애플리케이션입니다.

## 주요 기능

- **할 일 추가** — 텍스트 입력 후 `추가` 버튼 또는 `Enter` 키로 등록
- **완료 처리** — 체크박스 클릭 시 완료 상태로 전환, 완료 시각(YYYY-MM-DD HH:mm) 자동 기록
- **삭제** — 각 항목의 ✕ 버튼으로 개별 삭제
- **완료 항목 일괄 삭제** — 하단 `완료 항목 삭제` 버튼으로 완료된 항목 전체 삭제
- **필터 탭** — 전체 / 진행중 / 완료 탭으로 항목 분류 조회
- **남은 개수 표시** — 하단에 진행중인 할 일 개수 실시간 표시
